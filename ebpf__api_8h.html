)

Get list of programs and stats in an ELF eBPF file.

Parameters
[in]fileName of ELF file containing eBPF program.
[in]sectionOptionally, the name of the section to query.
[in]verboseObtain additional information about the programs.
[out]dataOn success points to a list of eBPF programs.
[out]error_messageOn failure points to a text description of the error.

The list of eBPF programs from this function is TLV formatted as follows:
sections ::= SEQUENCE {
section SEQUENCE of section
}

section ::= SEQUENCE {
name STRING
platform_specific_data INTEGER
count_of_maps INTEGER
byte_code BLOB
statistic SEQUENCE of statistic
}

statistic ::= SEQUENCE {
name STRING
value INTEGER
}
